# Сжатие изображений в JuyaShop

## Автоматическое сжатие при загрузке

Изображения автоматически сжимаются при загрузке через админ-панель:
- **Качество**: 85% (хороший баланс между размером и качеством)
- **Максимальный размер**: 1920x1920px (большие изображения автоматически уменьшаются)
- **Форматы**: JPEG, PNG, WebP
- **Оптимизация**: PNG без прозрачности конвертируются в JPEG для лучшего сжатия

## Сжатие существующих изображений

Для сжатия всех существующих изображений в папке `uploads/products`:

```bash
cd backend
npm run compress:images
```

Или с указанием пути:

```bash
cd backend
npm run compress:images путь/к/папке
```

Пример:
```bash
npm run compress:images uploads/products
```

## Сжатие статических изображений frontend

Для сжатия изображений в `frontend/public/images` можно использовать онлайн-сервисы или установить отдельные инструменты:

### Рекомендуемые онлайн-сервисы:
- https://tinypng.com/ - отличное сжатие PNG и JPEG
- https://squoosh.app/ - инструмент от Google с продвинутыми настройками
- https://imagecompressor.com/ - простое и быстрое сжатие

### Использование tinypng CLI (опционально):

```bash
npm install -g @tinypng/cli
tinypng frontend/public/images/*.png
```

## Результаты сжатия

После сжатия вы увидите:
- Размер до сжатия
- Размер после сжатия
- Процент экономии места
- Общую статистику по всем обработанным файлам

Типичное сжатие дает экономию **30-70%** размера файла без видимой потери качества.
